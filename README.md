# Arch Technologies Internship Program

## Machine Learning Internship – Month 1 & 2

This repository contains the completed tasks for the Machine Learning Internship.

---

## 📌 Task 1: Email / SMS Spam Classification
- Dataset: SMS Spam Collection
- Model: Multinomial Naive Bayes
- Feature Extraction: TF-IDF
- Accuracy: ~98%

### Files:
- Task1_Email_Spam_Classification.ipynb

## 📌 Task 2: MNIST Digit Recognition
- Dataset: MNIST
- Model: Logistic Regression
- Image Size: 28x28 pixels
- Accuracy: ~92%

### Files:
- Task2_MNIST_Digit_Recognition.ipynb

## 📌 Task 3: Housing Price Prediction
- Dataset: California Housing Dataset
- Model: Linear Regression
- Features: Income, Rooms, Population, Location, etc.
- Evaluation Metrics: R² Score, Mean Squared Error

### Files:
- Task3_Housing_Price_Prediction.ipynb

## 📌 Task 4: Iris Flower Classification
- Dataset: Iris Dataset
- Model: Logistic Regression
- Features: Sepal Length, Sepal Width, Petal Length, Petal Width
- Accuracy: ~95%+

### Files:
- Task4_Iris_Classification.ipynb
